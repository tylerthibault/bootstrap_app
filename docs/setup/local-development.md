# Local Development

## Prerequisites

- Node.js 20+
- npm 10+

## Initial setup

1. From repository root, install dependencies:
   - `npm install`
2. Verify baseline scripts:
   - `npm run lint`
   - `npm run test`
   - `npm run dev`

## Notes

- In Step 1, scripts are placeholders to verify command wiring.
- Real app-specific tooling and dependencies are added in later steps.
