# Expo Setup

## Location

- `apps/mobile`

## What was scaffolded

- Expo TypeScript app shell
- Android, iOS, and web platform config in `app.json`
- Scripts in `apps/mobile/package.json`:
  - `npm run start`
  - `npm run android`
  - `npm run ios`
  - `npm run web`

## Notes

- Dependencies are declared but not installed in this environment.
- Run install from repository root once Node/npm are available.
