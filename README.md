# smn_app_v1

Bootstrap monorepo for building cross-platform applications.

## Top-level structure

- `apps/mobile`: Expo mobile and web app-shell
- `apps/web-seo`: Public SEO site
- `apps/backend`: Flask API backend
- `packages/shared`: Shared cross-app code (types, utils)
- `packages/config`: Shared config presets
- `docs`: Implementation and operational docs

## Root scripts

- `npm run install`
- `npm run lint`
- `npm run test`
- `npm run dev`

These scripts are minimal placeholders in Step 1 and will be replaced with real tooling in later steps.
